# SerialPad · 串口坐标控制面板

一个基于 **Web Serial API** 的现代化网页应用，用于连接本地 COM 端口并通过键盘/鼠标发送控制命令。鼠标控制区域支持发送精确的坐标信息 `(x,y)`，适合需要位置控制的硬件项目（如机械臂、CNC、智能小车等）。

## ✨ 功能特性

- 🔌 **串口连接** – 搜索并连接本地 COM 端口，支持多种波特率（9600 ~ 115200）
- ⌨️ **键盘控制** – 使用 `WASD` 或方向键发送大写字母命令（U / D / L / R），按钮布局模拟键盘方向键
- 🖱️ **鼠标坐标控制** – 在 256×256 像素的方形区域内，**左键按下**发送相对于中心的坐标 `(x,y)`，范围 **-128～127**
- 🖱️ **鼠标点击控制** – 左键点击发送当前位置坐标；右键点击重置鼠标位置到中心并发送 `(0,0)`
- 📊 **实时日志** – 显示所有发送/接收的数据，支持暂停滚动和清空日志
- ⚙️ **自定义命令** – 可发送任意字符串命令，并提供“启动/停止/复位”快速按钮
- 🎨 **三栏布局** – 左侧键盘控制区、中央鼠标控制区、右侧命令统计区，界面清晰直观
- 📱 **响应式设计** – 自动适应不同屏幕尺寸（桌面、平板、手机）

## 📋 使用要求

- **浏览器**：Chrome 89+ 或 Edge 89+（需支持 [Web Serial API](https://developer.chrome.com/articles/serial/)）
- **运行环境**：必须通过 HTTPS 或 `localhost` 访问（本地开发无需证书）
- **串口设备**：任何通过 COM 端口连接的设备（如 Arduino、ESP32、STM32、PLC 等）

## 🚀 快速开始

1. **下载项目文件**：将 `index.html`、`style.css`、`script.js` 保存到同一文件夹。
2. **打开应用**：使用支持的浏览器打开 `index.html`。
3. **连接串口**：点击“搜索并连接 COM 端口”，在弹出的对话框中选择您的设备。
4. **选择波特率**（默认为 115200），确保与硬件匹配。
5. **开始控制**：
   - 使用键盘 `W/A/S/D` 或方向键发送方向命令。
   - 将鼠标移入中央方形区域，**按下左键**时发送坐标。
   - 左键点击发送当前位置坐标，右键点击重置位置。

## ⌨️ 键盘控制

键盘按钮已缩小并排列成类似方向键的十字布局，位于界面左侧。

| 按键         | 功能   | 发送命令 |
|--------------|--------|----------|
| `W` / `↑`    | 上方向 | `U`      |
| `S` / `↓`    | 下方向 | `D`      |
| `A` / `←`    | 左方向 | `L`      |
| `D` / `→`    | 右方向 | `R`      |

## 🖱️ 鼠标坐标控制

中央区域是一个带坐标刻度的 256×256 像素正方形，中心为原点 `(0,0)`。

| 操作                     | 功能                                                                 | 发送内容                |
|--------------------------|----------------------------------------------------------------------|-------------------------|
| **左键按下并移动**       | 当坐标变化超过阈值（默认 3 像素）时发送新坐标                        | `(x,y)` 格式，如 `(25,-30)` |
| 左键点击                 | 发送当前位置的坐标                                                   | `(x,y)`                 |
| 右键点击                 | 重置虚拟鼠标位置到中心                                               | `(0,0)`                 |
| 鼠标离开区域             | 自动发送中心坐标                                                     | `(0,0)`                 |

- **坐标系统**：X 轴向右为正，Y 轴向下为正（与网页坐标系一致）。
- **坐标范围**：X 和 Y 严格限制在 **-128 到 127** 之间，适合用于 8 位有符号整数传输。
- **视觉反馈**：移动时显示当前坐标标签和轨迹点，点击时有涟漪动画。

## ⚙️ 自定义命令

1. 在“自定义命令”输入框中输入任意字符串。
2. 点击“发送”按钮或按 `Enter` 键。
3. 可使用快速按钮发送预设命令（启动、停止、复位）。

## 🔧 配置说明

### 修改波特率
在串口控制区域的下拉菜单中选择合适的波特率，默认为 `115200`。

### 修改键盘命令映射
在 `script.js` 的 `commandMapping` 对象中修改：

```javascript
this.commandMapping = {
    'up': 'FORWARD',    // 改为 "FORWARD"
    'down': 'BACKWARD',
    'left': 'TURN_LEFT',
    'right': 'TURN_RIGHT',
    'START': 'START',
    'STOP': 'STOP',
    'RESET': 'RESET'
};
```

### 调整坐标发送阈值

在 `constructor` 中修改 `coordSendThreshold`（单位：像素）：

```javascript
this.coordSendThreshold = 3;  // 数值越小越灵敏，越大越省频
```

### 添加快捷键

在 `keyMapping` 对象中添加或修改键盘映射：

```javascript
this.keyMapping = {
    'w': 'up', 'ArrowUp': 'up',
    's': 'down', 'ArrowDown': 'down',
    'a': 'left', 'ArrowLeft': 'left',
    'd': 'right', 'ArrowRight': 'right',
    ' ': 'STOP',   // 空格键发送 STOP
};
```
